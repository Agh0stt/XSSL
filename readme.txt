Official Guide From Abhigyan Ghosh::
for first install do:
chmod +x ./build.sh
then do::
./build.sh
it will download XSSL;
them it will give a list of options.
to start interpreter (repl mode) do::
./START
to run a .xssl file do::
./xssl-run <filename.xssl>
for example:'
./xssl-run simple.xssl
.........

Good luck
